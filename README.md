# 整うCLOUD - プロジェクト概要

「書類も、サブスクも、生活も。全部、整う。」をコンセプトにした、個人向け管理サービスのランディングページ（LP）です。複雑な生活管理をシンプルに整理する、クリーンで直感的なデザインを目指しています。

---

## 🚀 特徴

* **自動仕分け機能**: AIが書類の種類を判別し、自動でカテゴリー分けを行います。
* **リマインド通知**: サブスクリプションの更新日や重要書類の期限を事前にお知らせします。
* **一元管理**: 保険、税金、賃貸契約など、バラバラになりがちな情報をひとつの場所に集約。
* **レスポンシブ対応**: PCとスマートフォン、どちらからでも美しく、使いやすいUI/UXを提供。

---

## 🛠️ 使用技術

* **HTML5 / CSS3**: セマンティックなマークアップとモダンなレイアウト。
* **JavaScript**: スムーズなスライダー機能、オーバーレイメニュー。
* **Adobe Fonts**: 視認性とブランドイメージを両立させたタイポグラフィの採用。
* **SVG Graphics**: 高精細（Retina）ディスプレイでもぼやけない、軽量でクリアなビジュアル。

---
